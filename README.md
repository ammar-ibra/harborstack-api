# HarborStack API - Quick Start

### (Setup)

1. **تثبيت المكتبات:** `npm install`
2. **إعداد البيئة:** انسخ ملف `.env.example` إلى ملف جديد باسم `.env`.
3. **التشغيل:** `npm run dev`
4. **المنفذ (Port):** السيرفر يعمل على المنفذ المحدّد في ملف الـ `.env` (افتراضياً `3000`).

### أمثلة للروابط (Example URLs)

- **جلب الطواقم:** `GET http://localhost:3000/api/v1/crews`
- **جلب المناوبات:** `GET http://localhost:3000/api/v1/shifts`
